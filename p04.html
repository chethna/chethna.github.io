SCRIPT TO CONVERT SIM DATA TO SKINNING


About

During the summer of 2015, I was interning at NexRef Technologies. I was creating 3D content for their Augmented reality app - XR. I would usually create the model in Maya, and import it into Unity. One of the projects required me to create a nCloth simulation in Maya and take it back into Unity, simple right? My thoughts exactly. Create the simulation in Maya and simply import it into Unity, but the only problem (big problem) was unity doesn't read simulation data! Like I said, Unity doesn't read simulation data, but it does read rigging(joints) data. So I wrote a script to convert nCloth simulation into skinned mesh. The script also handles baking the data into key frames which can be then be exported to any game engine.

Maya, Unity, Python

Update 3

Chethna Kabeerdoss

Procedural modeling of 3D geometry using L-systems

Summary of Work to Date – After my second update, which had the artist's work that I wanted to replicate, I was getting my self familiar with turtle graphics and using it in Houdini. I was a little worried based on the remarks I got for update 2. Dr. Schaefer was concerned as to how I was going to render the geometry, and we decided to use Houdini to tackle this issue.

Implementation : I picked a tree I thought would be easy to start with my beginner's knowledge of turtle graphics.






Analysis of Work – I started off with just writing rules for the base structure of the tree. The leaves of the tree are like repeating circles (squiggles). For this I constructed a rule set separately and joined it in between the rules for the base structure of the tree.





Plan for Completion – I maybe a bit behind schedule, but I think I should be able to get up to 3 trees - fully rendered for the completion of this project. I am going to work on this during the thanksgiving break, which I hope will help me finish a major portion of this project.
